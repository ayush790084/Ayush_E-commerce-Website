## ⚙️ Installation & Setup

Follow these steps to run locally:

```bash
# Clone the repo
git clone https://github.com/ayush790084/Ayush_E-commerce-Website.git

# Go to project folder
cd Ayush_E-commerce-Website

# Install dependencies
npm install

# Run the project
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## 🌐 Deployment

Deployed on **Vercel**

```bash
npm install -g vercel
vercel
```
